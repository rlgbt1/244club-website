# 244 Club — Website

**Live:** [244club.com](https://244club.com)

---

## What is 244 Club?

244 Club is a curated society for Angolan students and young professionals in the United Kingdom. The name comes from **+244** — Angola's international dialling code — a symbol of connection to home, cultural identity, and community across distance.

The club operates across two pillars: **social & cultural** (dinners, meetups, cultural gatherings) and **professional & development** (industry talks, networking, and partnerships — including a flagship event with McKinsey & Company).

---

## The Website

A custom-built presentation and sign-up website, designed and developed from scratch as the club's public-facing hub. It replaces a basic Linktree and serves as the brand's permanent home online.

**Live at** `244club.com`, hosted on GitHub Pages.

---

## Built with

- **Pure HTML, CSS, and vanilla JavaScript** — no frameworks, no build tools
- **Space Grotesk** for UI typography; **Playfair Display** as an italic serif accent on key display words
- **Scroll-driven orbital animation** — four circular event photos rotate around the central logo as the user scrolls, built with a `requestAnimationFrame` loop and CSS transform
- **Tally** for form handling and sign-up data collection
- **GitHub Actions** for continuous deployment to GitHub Pages

## Design

Warm ivory backgrounds with deep forest green as the primary brand colour. Angola's flag colours (red and gold) appear as subtle typographic accents. The design is mobile-first and built for a young, professional audience.
